# PLMScoRe
R package for deriving PLMS indices from leg movement information according to the newest WASM 2016 rules
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.3903318.svg)](https://doi.org/10.5281/zenodo.3903318)
